Solving First Order ODE


Radioactivity Decay

Radioactive Decay is a phenomenon wherein heavy unstable atoms decomposes into smaller more stable atoms by releasing excess energy as gamma radiation. Radioactive decay is an inherntly random process and it can not be predicted exactly which atom will decay at an instantaneous moment

We can represent this rate of change as a differential equation

$$ \frac{dN}{dt} = -kN$$ Questions related to numerical analysis should always be solved analytically first for better understanding. Thus,

Analytic Solution

According to Radioactive law of decay: $$ \frac{dN}{dt} \propto N$$ Replacing the proportionality constant with '-k' $$ \frac{dN}{dt} = -kN$$ $$ \frac{dN}{N} = -kdt$$ $$ \int_{N_0}^{N} \frac{dN}{N} = -\int_{0}^{T}kdt$$ $$ \int_{N_0}^{N} \frac{dN}{N} +k\int_{0}^{T}dt = 0$$ $$ \log_{e}(\frac{N}{N0}) +kT = 0$$ Taking natural logarithm on both sides $$ N = N_0e^{-kT}$$

Thus we can see that solution of the differential equation is an exponential function which is raised to the power of -kT. Evaluating such functions before hand allows us to check if the program is working properly. Then we can apply the program to find the numerical solution of some differential equation to find out the nature of solution.

Scilab Program

Now, we'll use scilab to solve odinary differential equations.We will use the inbuilt ode() function which solves explicit odinary differential equations, defined by $$ \frac{dy}{dt} = f(y,t)$$ such that $$y(t_0) = y_0$$ Though the knowledge of the numerical methods being used is not required for further information you can always do further reading from scilab's official website.

The following is the program for calculating the solution of differential equation of radioactive decay of Uranium atoms


				k = 4.96*10^(-18);
				N0 = 1000; 
				T0 = 0;
				T = linspace(0,10^18,100);
				function dN = f(T,N)
				    dN = -k*N
				endfunction
				n = ode(N0,T0,T,f); 
				plot(T,n)
				xlabel("time(sec)")
				ylabel("Number of Atoms")
				title("Solution of Radioactive Decay of Uranium atoms")
For understanding better the whole program can be sub-divided.

The first step is initialising variables. Here 'k' stands for the radioactivity decay constant,'N0' is the initial number of atoms, 'T0' is the time from which values are being recorded.
Here, linspace is a speacial function that is used to create an array of values from value given in the first parameter to the value in second parameter. The number of values in between will be equal to the value given in as the last parameter


				k = 4.96*10^(-18);
				N0 = 1000; 
				T0 = 0;
				T = linspace(0,10^18,100);

The most important part of the program is the function: 


				function dN = f(T,N)
				    dN = -k*N
				endfunction
We are basically expressing the differntial equation in the syntax of scilab function.



				n = ode(N0,T0,T,f); 

				plot(T,n)
				xlabel("time(sec)")
				ylabel("Number of Atoms")
				title("Solution of Radioactive Decay of Uranium atoms")

The first line of the program calls the ode function which solves the given function for the differential equation. The rest of the program plots the values and draws the required legends,etc.
